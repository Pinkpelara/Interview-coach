# Seatvio Prompt Asset: Question Bank Generation

Generate a personalized interview question bank from:
- Resume timeline and achievements
- Job description requirements and values language
- Alignment gaps and likely probe areas

## Required categories and minimum counts
- Behavioral: 6
- Technical/Functional: 4
- Situational: 4
- Company-Specific: 3
- Curveball: 3
- Opening: 2
- Closing: 3

For each question provide:
1. question_text
2. why_asked (2-3 sentences)
3. framework (STAR/SOAR/PAR/PREP/direct)
4. model_answer (first person, specific)
5. what_not_to_say
6. time_guidance_seconds
7. likely_followup
8. difficulty (1-5)
